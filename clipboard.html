Method: copy
Arguments:
selector, ?soughtAttribute, ?trim
Returns:
true or false

Important for copy method:

Document should contain the selector (selector must be a string, Node, NodeList, HTMLCollection, HTMLAllCollection or jQuery element)!

Selector should contain some info!

If soughtAttribute was specified but not found in DOM it'll be counted as unspecified!

Argument trim determines would whitespaces be removed from the sides of a copied text.

Also soughtAttribute can be instead of trim in call.

Examples:
						clipboard( 'input' );
						clipboard( 'div', true );
						clipboard( 'img[name="target"]', 'src', true );
						clipboard( jQuery( 'span[role="alert"]' ) );
						clipboard( document.querySelectorAll( 'textarea' ), undefined, true );
Method: insert
Argument:
?string
Returns:
true or false

Important for insert method:

If string argument was passed as empty string or undefined it will work as clear method.

Example:
						clipboard( 'insert', 'String to be inserted' );
Method: clear
Returns:
true or false
Method clears user's clipboard.
Example:
						clipboard( 'clear' );
Method: isSupported
Returns:
true or false
Method verifies is browser supports functionality which is necessary for proper operations.
Example:
						clipboard( 'isSupported' );

Usage:

clipboard( method );

Calls the method if it was found.

clipboard( method, ...?mixed );

Calls the method if it was found with ...?mixed arguments.

Otherwise method copy will be called with selector (also soughtAttribute and trim arguments if they were specified).

clipboard( mixed );

If the first argument of the mixed was not found as a method, the copy method with the mixed arguments passed will be called.

? — means the argument isn't required.
